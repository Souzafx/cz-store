Esta pasta deve conter o arquivo xlsx.full.min.js (biblioteca SheetJS)
para que a funcionalidade de importar planilhas Excel funcione.

Para baixar (faça uma vez, depois funciona offline):

1) Abra o Terminal
2) Execute:

   cd "/Users/davyd.dg12gmail.com/Documents/CZ automação/lib"
   curl -L -o xlsx.full.min.js https://cdn.sheetjs.com/xlsx-0.20.1/package/dist/xlsx.full.min.js

Pronto! Agora a importação de Excel vai funcionar.

Se você não usar importação de Excel, pode ignorar este passo —
todo o resto do sistema funciona normalmente sem esse arquivo.
